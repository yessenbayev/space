# Space Game

Needs Python 3+ to run the game.  

This program is intended as a learning exercise and in no way is a complete game.  
Please install Pygame before running the program.  
`pip install pygame`

or

`pip3 install pygame`

Run space.py to play the game.  
`python space.py`

## Controls
1. WASD = Movement
2. Space = Shoot
3. Esc = Exit program
